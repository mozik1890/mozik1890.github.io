短发

发表于 2015-05-07   |   分类于 闲言碎语   |  

written on 2009.10.22

周二的选修课上有一个短发，一度是我的向往。我喜欢她，我想。这没有什么不好意思的。我不知道她的名字，甚至没看清她的面孔，但这并不妨碍我喜欢。当然，我们没有言语，也不需要。沉默是最好的语言。静静地看着她，我感到以往岁月中的好几个影子又在眼前浮现了，它们重叠在一起，相互纠缠着我的记忆，让我不知所措，不知所想。你也许也会有这样的经历：在拥挤的大街上，在飞速的列车上，也许突然有那么一个影子在你眼前一闪而过，让你的心突地一动，你想开口说些什么但你什么也说不出，你脑海中复习浮现出许多片段让你不知道在想些什么，你就是向往，就是想多看一眼，可在你的发愣间，伊人已经消失在茫茫人海……­

就是这样，我喜欢她，没有必要害羞什么。尽管她今天没有出现，但如果真的再次遇见，我们还会是沉默，我甚至还看不清她的面孔，但这不会妨碍我在人群中一眼认出她。我什么也不会说，她什么也不需知道，她就像是那个装饰我的梦的人，她尽管自看她的风景，而我，也自己继续自己的梦，如此简单，如此让人留恋.

我想我是地球

发表于 2015-05-07   |   分类于 闲言碎语   |  

written on 2009.03.30

今天忽然想我是地球！嘿，比尼采逊多了！原因嘛，听我道来！

我想我是地球，每天围着自己自转。从白天到黑夜，从黑夜到白天。我愿意按时地起床，吃过饭以一种不快不慢的速度随着人流去教室，我愿意一个人坐在角落不声不语直到下学，我愿意不发短信不打电话没人理我…我愿意一个人自转不要流星冲撞！我想我真的承受不起！­

我想我是地球，有着缤纷的生活，只是也隔了厚厚的一层臭氧。从来我的影响范围就只是这一小圈，前后左右而已。但我有我的生活，酸甜苦辣，应有皆有。我有很多的想法，让它在脑海里自由发展，没人知道，也不需要人知道，顺其自然，也许才会有文明出现。­

我想我是地球，也许我有天会突然找到自己的太阳，然后义无反顾的围着公转。她会给我春夏秋冬，给我快乐悲伤。我的生活因她而发光，为她哭为她笑，就算她只是给我十亿分之一得热量！­

我想我是地球，我肯定也会有月亮哈！默默地在我的身影下转动，不声不响。没有人注意却潜移默化了我。在我无依无靠的时候我会想到他，走的再远他也在身旁。­

我想我是地球，我默默地转了这么多年，还会继续的一如既往地转下去….

时间 空间

发表于 2015-05-07   |   分类于 诗情   |  

written in 2009.10.14

我身处一个莫名的空间
一天长过一年|
在没有你的时间
我总觉的特别孤单
从来不开口要求什么
沉默最是能勾起思念
直到所有的微笑都成为掩饰
直到所有的泪水都风化不见
距离从来不会阻隔
为何分开却成了爱情的终点

我身处一个莫名的时间
日子忽而就回到了从前
说过的话语依然清晰
牵过的手也依然温暖
也许不曾有过这些
也许不曾有过从前
岁月不会抹去什么
可是什么让我把你遗忘
成了云烟
